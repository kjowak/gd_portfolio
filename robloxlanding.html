Roblox Development

My most Advanced Work. Fueled by 7 years of experience.

"Powering Imagination"

My most valuable experiences in game development come from the platform Roblox, which taught me concepts such as the client-server model, event-based programming, and object-oriented design. I learned these concepts through doing, attempting to create various systems and prototypes for the many game ideas I had. This page highlights a few examples, which you can find more about by clicking the "Learn More" button next to each project.

Blackjack

My goal with this project was to create an online version of my favorite card game: Blackjack. This project comes in 2 iterations, the original being created in Fall 2020, and the newest version being created over the 2022-2023 winter break. The original version features a chip system, where players can bet and earn chips. The newer version features a juiced-up user interface, with actual cards being displayed instead of being a text-based interface.

Weapon Systems

Shooter games are arguably the most popular genre of video game, so I decided to try my hand at making my own. This led me to create weapon systems, which allow developers to easily create new weapons as they're needed by simply providing the needed configurations. Again, I have 2 versions here, with the older one being far less efficient, although more complete, than the newer system.

Star Wars Simulator

Star Wars Simulator was a game my friends and I wanted to create during the Summer of 2022. Although it never came to full completion, I was able to prototype several systems that would've been used in the final product, such as a weapon system that supports both ranged and melee weapons, force abilities, and a leveling system. This game was also one of my first attempts at saving player data, which included the player's inventory and level.
